## 🚀 Featured Project — [Floci](https://floci.io)

> **A fast, free, MIT-licensed AWS service emulator built with Quarkus and compiled to a native binary via GraalVM.**

Floci is a lightweight alternative for local AWS development — no paywalled features. If you work with AWS locally, give it a try.
